# 💫 About Me:

Hey there!  
I'm a passionate developer who loves mixing **code and creativity**. Right now I'm diving deep into **game development** with **Godot Engine** (learning **GDScript**) while also exploring **3D modeling** with **Blender**.  

💡 I enjoy experimenting with different languages and frameworks, from **low-level stuff (C/C++)** to **high-level scripting (Python/JavaScript)**.  
🎮 My current focus: building my own **indie game project** from scratch.  
⚡ Always curious, always learning — whether it’s coding, design, or trying new tools.  

When I’m not coding, you’ll probably find me tweaking some 3D models, testing game mechanics, or just breaking things to see how they work.
